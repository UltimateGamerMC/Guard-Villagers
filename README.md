# Guard Villagers

Fabric mod for Minecraft **26.1.1**: villagers can become armed guards, defend villages, use ranged weapons, and interact with players (inventory, follow, patrol). Convert a villager by sneaking and using a sword or crossbow on a nitwit or unemployed adult villager (configurable Hero of the Village requirement).

**Build:** `./gradlew build` — jar in `build/libs/`. **Dev server:** `./gradlew runServer` (accept EULA in `run/eula.txt`; change `server-port` in `run/server.properties` if 25565 is in use).

Requires Fabric Loader, Fabric API, and Java 25.
